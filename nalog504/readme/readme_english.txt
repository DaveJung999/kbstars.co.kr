*** INSTALLATION

Firstly, you should make sure that your host supports PHP script execution and
MySQL database access. Also, you should have obtained information about your
MySQL database connection (DB HOST, DB USER ID, DB PASSWORD, DB NAME).

Secondly, upload your copy of n@log 5 with an named folder (e.g. nalog5) to your
FTP server (including all the subfolders). Set the permission of this folder to
707 or 777. Then, you can execute install.php in the program folder to begin the
installation process:

i.e. http://Site_URL/n@log_Folder/install.php

The program will be successfully installed if you provide correct information of
MySQL database connection. Next, you can login as administrator to create and
configure your counter(s).

(Notice: If you are unable to install, please check if the permission value of
n@log 5 's program folder is set correctly - 707 or 777)


*** UPGRADING INSTRUCTIONS

You can easily upgrade from n@log version 3 or 4 to version 5. After overwriting
your n@log folder with all the files (including subfolders) supplied with
n@log 5, then you can execute upgrader.php in the program folder to begin
the upgrade process:

i.e. http://Site_URL/n@log_Folder/upgrader.php

(Notice: If you are upgrading from n@log 3, some of your counters' statistics -
by hours, days, weeks, months, years - will not be converted.)
